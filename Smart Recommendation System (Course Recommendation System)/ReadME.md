# Smart Recommendation System

## Description
A Machine Learning based Course Recommendation System that recommends courses based on user interests and skill levels.

## Features
- Personalized course recommendations
- Interest-based recommendations
- Skill-level based suggestions
- Recommendation accuracy evaluation
- Course popularity visualization
- Course domain distribution analysis
- Machine Learning recommendation model

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib

## How to Run
pip install -r requirements.txt

python course_recommendation.py
