AI-Based Fake News Detection Tool
Overview

The AI-Based Fake News Detection Tool is a Machine Learning project that identifies whether a news article or headline is Real or Fake using Natural Language Processing (NLP) techniques. The system processes text data, extracts meaningful features using TF-IDF vectorization, and classifies the news with a trained Machine Learning model.

Features
Detects fake and real news articles
Text preprocessing using NLP
TF-IDF feature extraction
Machine Learning-based classification
Saves the trained model for future predictions
Easy-to-use command-line interface
Technologies Used
Python
Pandas
Scikit-learn
TF-IDF Vectorizer
Multinomial Naive Bayes
Joblib
Project Structure
main.py – Main application
news.csv – Dataset
requirements.txt – Required libraries
README.md – Project documentation
